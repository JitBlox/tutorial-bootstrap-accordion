A data-bound Bootstrap 5 Accordion with Angular

This project is part of the JitBlox tutorial  Building a data-bound Bootstrap Accordion . Check out the tutorial for more information. Feel free to save a copy of this project and adapt it to your own needs.

After you successfully complete the tutorial, the  characters page  displays a list of  Game of Thrones  characters in a Bootstrap Accordion, linking to a detail view of each character.
